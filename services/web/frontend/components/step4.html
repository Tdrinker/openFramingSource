Build a Frame Classification Model with Deep Learning

The goal of Step 4 is to use the documents you have labeled from Step 3 to build a frame classification model. OFAI’s analytical framework incorporates the state-of-the-art deep learning model BERT, which stands for Bidirectional Encoder Representations from Transformers. More specifically, we use a recent extension of BERT: XLM-Roberta to analyze text in multiple languages. Here we’re using XLM-Roberta to classify text examples into categories, i.e. frames. This is a supervised method, meaning we will need a sample of labeled documents to train the model.
